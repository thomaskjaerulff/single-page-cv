# Single-Page CV

This project is a simple **single-page CV (resume)** built using basic web technologies.  
It is part of the **Roadmap.sh Frontend Projects** learning path.

🔗 Project reference: https://roadmap.sh/projects/single-page-cv

---

## 📄 Project Overview

The goal of this project is to create a clean, semantic, and accessible CV presented on a single web page.

### Key objectives
- Practice writing **semantic HTML**
- Apply **basic CSS layout and styling**
- Structure content clearly for readability
- Build a simple project suitable for a portfolio

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (no frameworks)
- Optional: Google Fonts

---

## ✅ Features

- Single-page layout
- Clear sections for:
  - Personal information
  - Skills
  - Experience
  - Education
- Responsive design
- Semantic HTML structure
